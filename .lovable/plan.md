

## Enable Auto-Confirm Email Signups

Enable auto-confirm so you can sign up and test the Meta connect flow immediately without waiting for a verification email.

### What changes

1. Use the `cloud--configure_auth` tool to enable auto-confirm for email signups
2. No code changes needed

### After this

You'll be able to:
1. Sign up on `/auth` with any email/password
2. Immediately access the wizard
3. Click "Connect with Meta" to test the OAuth flow
